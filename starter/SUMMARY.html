TTS Voice Cloning Assignment Summary

Objective

The objective of this assignment was to generate a style tensor that closely matches the target speaker using the provided Kokoro TTS model. The model weights remained frozen throughout the process, and only the voice embedding tensor was optimized.

Methodology

  • Evaluated all provided stock voices using the supplied speaker similarity metric.
  • Generated the provided baseline blend using the two highest-scoring stock voices.
  • Modified search.py to perform stochastic optimization with multi-sentence evaluation, structured tensor perturbations, adaptive step sizes, and simulated annealing acceptance.
  • Compared the optimized tensor against the baseline using the same similarity metric.

Experiments

Several optimization iterations were performed starting from the baseline blend. While the search explored nearby regions of the embedding space, none of the generated candidates consistently exceeded the baseline similarity score.

Results

The baseline blended voice achieved the highest similarity score during experimentation (approximately 0.6331). Since the optimization search did not improve upon this score, the baseline tensor was selected as the final submission and saved as voice.pt.

Conclusion

The assignment demonstrates that simple blending of high-quality pretrained voices provides a strong initialization. Although stochastic local search explored additional candidate embeddings, the baseline remained the best-performing solution within the available time budget.
